Confidential Transactions

Transacted amounts on the Bitcoin Phiber blockchain are blinded from the public. Blinded amounts are cryptographically protected by range proofs, and allow two participants to transact privately while cryptographically guaranteeing that no more coin is spent than is available. While only the participants of a transaction have access to the key used for unblinding amounts, an unblinding key can be provided to a trusted third party for accountability and auditing purposes.

Security

To ensure the blockchain is protected in a post-quantum future, Bitcoin Phiber exclusively make use of the NIST FIPS 202 SHA-3 final standard for all of its hashing computations. This includes our mining hash challenges, transaction Merkle trees, witness commitment Merkle trees and even user address checksums.






SegWit

All transactions on the Bitcoin Phiber network employ the BIP141 Segregated Witness protocol. Aside from eliminating the problem of transaction malleability, SegWit reduces transaction size to increase the number of transactions per block. These advantages enhance security and improve the scalability of the network.
